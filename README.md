# Завдання 1: Аналіз заробітних плат

## Опис завдання

Розробити функцію `total_salary(path)`, яка аналізує текстовий файл зі інформацією про місячні заробітні плати розробників та повертає загальну та середню суму заробітної плати.

## Формат даних

Файл містить кілька рядків, де кожен рядок має формат:
```
Ім'я Розробника,Заробітна плата
```

Приклад:
```
Alex Korp,3000
Nikita Borisenko,2000
Sitarama Raju,1000
```

## Використання

```python
from task_1 import total_salary

# Повернення результату
total, average = total_salary("salary.txt")
print(f"Загальна сума заробітної плати: {total}, Середня заробітна плата: {average}")
```

Очікуваний результат:
```
Загальна сума заробітної плати: 6000, Середня заробітна плата: 2000.0
```
